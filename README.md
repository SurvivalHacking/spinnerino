# Spinnerino
Un giro nel passato

## 📘 Descrizione

Mini arcade dedicato ai titoli con controllo a manopola con grandi classici come Arkanoid. Un concentrato di retro-gaming e DIY in un formato unico.
Un progetto di Marco Prunca basato su ESP32-P4.
